Roadmap

Vidar has several epics planned for the next phase of development. These will most likely continue for multiple minor releases. If any of them interest you, feel free to contribute!

True Node Support

We are currently working on a vidar-node rewrite. The rewrite will use Node implementations of browser technologies to create a seamless Node integration without running in a headless browser.

Audio System

A new audio system, with more audio layers and the introduction of audio effects is on its way!

Some of the highlights:

  • Panner effect
  • Equalizer effect
  • Compressor effect
  • Echo effect
  • Reverb effect
  • Pitch shift effect

Layer Inheritance

A new group layer will be introduced. A group layer will have both audio and visual support, along with multiple children.

Transitions

A transition system is also on its way! You will be able to attach any built-in or custom transition to two layers. The transition will then interpolate the audio and/or visual output of both layers.

3D System

Although still in early planning, we could use three.js to add a 3D mode to movies, where every visual and audio layer has a 3D position. Alternatively, we could get rid of the concept of a "3D movie" and allow both 2D and 3D layers to be added to any movie. Feel free to post on the GitHub discussion if you have any ideas!

Other

Other planned changes:

  • Creating sources directly from path / url
  • Duplicating layers
  • Improved gaussian blur
  • More visual effects
